## Office stationery

Application for manage office stationery. 

## Laravel Framework

The laravel documentation can be found here [Laravel official website](https://laravel.com).

## Installation

- Clone this repository
- Run `composer install`
- Run `php artisan key:generate`
- Copy .env.example file in the root of the project and change it's name with .env
- Run `php artisan key:generate`
- Create database and specify that database name into variable name called `DB_DATABASE=your_db_name`
- Run `php artisan serve` to start the application
